# 常量定义及其含义

## 数据库部分

#### 币种

currency code

> 0001 - RMB

#### 商品状态

commodity state code

> 0001 - 待售
>
> 0002 - 下架
>
> 0003 - 被购买
>
> 0004 - 被退还

>
